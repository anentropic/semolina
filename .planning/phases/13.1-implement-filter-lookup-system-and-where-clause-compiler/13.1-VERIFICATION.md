## Re-verification Context

The initial verification (22/22 must-haves) was run before UAT. UAT (`13.1-UAT.md`) surfaced 3 minor issues:

1. `Field.__ne__` produced `Not(Exact(...))` compiling to `NOT (field = value)` — should be `field != value`
2. `In` and `Between` Lookup generic type args were bare `Any` — should express structural constraints
3. `filtering.md` was missing the `where(None)` conditional one-liner pattern and showed a trivial escape hatch example

Plan 05 (commits `886536e`, `7994abd`) closed all three. This re-verification checks those 3 new truths plus does a regression check on the 22 original truths.
