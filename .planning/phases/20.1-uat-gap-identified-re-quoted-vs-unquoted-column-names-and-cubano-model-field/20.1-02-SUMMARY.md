## Decisions Made

- `normalize_identifier()` is abstract on `Dialect` — each dialect knows its identifier folding convention. MockDialect uses identity so all existing MockDialect-based tests remain green.
- `source_name` on `IntrospectedField` is `None` for standard Snowflake UPPERCASE columns (ORDER_ID → lower → "order_id" → upper → "ORDER_ID" round-trips correctly). Set only for quoted-lowercase columns where `python_name.upper() != original_col_name`.
- `data_type=None` and `data_type.startswith("TODO:")` both produce "Any" in the rendered output. The distinction (None vs TODO) is preserved in `_FieldContext.todo_comment` for the `# TODO:` comment generation.
- WHERE predicates store Python field_name strings. `normalize_identifier` is applied at compile time in `_compile_predicate`, not stored in the predicate IR.
