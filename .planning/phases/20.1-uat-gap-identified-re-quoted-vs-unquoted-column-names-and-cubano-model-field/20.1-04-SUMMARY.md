## Decisions Made

- **Two-pass ruff pipeline:** `ruff format` handles whitespace/style (Black-equivalent), then `ruff check --fix --select I` handles import sorting. Both are independent with fallback — format failure returns original source; isort failure returns formatted-but-unsorted source.
- **Typer epilog `\n\n` trick:** Typer collapses single `\n` within a paragraph to a space (matching Typer source). Double `\n\n` creates a new paragraph that is rendered as a separate line. Each exit code row uses `\n\n` as separator.
- **Rich color convention:** green=0 (success), yellow=1/2 (user-fixable errors), red=3/4 (system/connection errors). Matches common CLI conventions (git, docker).
- **Ruff isort alphabetizes from-import names:** ruff isort sorts `from cubano import ...` names alphabetically, so the CLI test assertion was updated to expect `Dimension, Fact, Metric, SemanticView` (alphabetical) rather than template order.
