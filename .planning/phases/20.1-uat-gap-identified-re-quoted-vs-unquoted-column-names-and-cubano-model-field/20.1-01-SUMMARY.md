## Decisions Made

**`__get__` overload form:** `-> Field[T] | T` in the implementation signature is the only form that satisfies both `@overload` branches under basedpyright strict mode without `type: ignore`.

**pyproject.toml exemptions:** Adding `reportUnknownMemberType = false` etc. is the correct pyproject.toml-level approach for Python 3.11/3.12 where `TypeVar` defaults (PEP 696) don't exist. This is explicitly the pattern for ORM-style Generic descriptor libraries.

**`source=` in repr:** Showing it only when set keeps existing repr tests unchanged while providing useful debug output for the case that matters.
