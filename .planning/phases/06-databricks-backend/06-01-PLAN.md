  "databricks-sql-connector is required for DatabricksEngine. Install with: pip install cubano[databricks]"

**to_sql(query: Query) -> str:**
- Reuse SQLBuilder from sql.py with DatabricksDialect
- Pattern: SQLBuilder(self.dialect).build_select(query)
- This generates SQL with MEASURE() wrapping and backtick-quoted identifiers
- No new code needed — just delegation to existing Phase 3 implementation

**execute(query: Query) -> list[dict[str, Any]]:**
- Import databricks.sql (already imported in __init__, available in scope)
- Use context manager for connection: with databricks.sql.connect(**self._connection_params) as conn:
- Nested context manager for cursor: with conn.cursor() as cur:
- Generate SQL: sql = self.to_sql(query)
- Execute: cur.execute(sql)
- Get column names from cursor.description: columns = [desc[0] for desc in cur.description]
- Fetch results: rows = cur.fetchall()
- Map tuples to dicts: return [dict(zip(columns, row, strict=True)) for row in rows]
- Wrap in try-except to catch databricks.sql.exc errors (DatabaseError, OperationalError, Error)
- Translate to RuntimeError with helpful message

**Error handling:**
- Import from databricks.sql.exc: DatabaseError, OperationalError, Error
- Catch DatabaseError (SQL syntax, invalid objects): raise RuntimeError(f"Databricks query failed: {e}") from e
- Catch OperationalError (connection, permission): raise RuntimeError(f"Databricks operational error: {e}") from e
- Catch Error (generic fallback): raise RuntimeError(f"Databricks error: {e}") from e

**Docstrings:**
- Class docstring explaining connection parameters, MEASURE() syntax, Unity Catalog three-part naming, and context manager pattern
- Include example showing connection parameter dict with server_hostname, http_path, access_token
- Document that http_path must reference SQL warehouse in format /sql/1.0/warehouses/{warehouse_id}
- Document that Unity Catalog three-part names (catalog.schema.view) work transparently through DatabricksDialect
- Each method with comprehensive Args/Returns/Raises sections
