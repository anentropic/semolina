3. Create OrderTerm as a frozen dataclass AFTER the Field class definition:

```python
@dataclass(frozen=True)
class OrderTerm:
    """
    Wrapper specifying sort direction for a field in order_by().

    Created via Field.asc() or Field.desc() methods.
    """
    field: Field
    descending: bool = False

    def __repr__(self) -> str:
        direction = "DESC" if self.descending else "ASC"
        return f"OrderTerm({self.field.name}, {direction})"
```

4. Add .asc() and .desc() methods to the Field class:

```python
def asc(self) -> OrderTerm:
    """Return ascending OrderTerm for use in order_by()."""
    return OrderTerm(field=self, descending=False)

def desc(self) -> OrderTerm:
    """Return descending OrderTerm for use in order_by()."""
    return OrderTerm(field=self, descending=True)
```
