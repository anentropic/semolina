  <files>src/cubano/fields.py</files>
  <action>
Create the field descriptor module with Field base class and three subclasses (Metric, Dimension, Fact).

Implementation details:
- Field base class with descriptor protocol (`__set_name__`, `__get__`, `__set__`, `__delete__`)
- `__set_name__` performs validation:
  - `str.isidentifier()` to validate identifier syntax
  - `keyword.iskeyword()` and `keyword.issoftkeyword()` to check Python keywords
  - Reject reserved names that collide with dict methods: 'keys', 'values', 'items', 'get', 'pop', 'update', 'clear'
- `__get__` returns self for class-level access (Sales.revenue), raises AttributeError for instance access
- `__set__` and `__delete__` raise AttributeError to prevent modification
- Metric, Dimension, Fact subclasses (empty bodies, inherit Field behavior)
- Store name and owner attributes in Field instance

Use patterns from RESEARCH.md Pattern 1 and Pattern 4.
