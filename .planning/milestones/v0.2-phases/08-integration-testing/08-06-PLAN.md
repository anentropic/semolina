**Helper function: _create_credentials_env_file()**
```python
def _create_credentials_env_file(tmpdir: Path, filename: str, **vars):
    """Create .env file with given variables in tmpdir."""
    env_file = tmpdir / filename
    content = "\n".join(f"{key}={value}" for key, value in vars.items())
    env_file.write_text(content)
    return env_file
```

Use pytest's tmp_path fixture and monkeypatch for environment variable manipulation.
