## Decisions Made
- Predicate base class is a plain class (not a dataclass) with `__and__`, `__or__`, `__invert__` operator methods
- All 15 lookup subclasses use empty class bodies (`class Exact(Lookup[Any]): ...`) inheriting fields from Lookup
- IsNull uses `Lookup[bool]`, Like/ILike/StartsWith/IStartsWith/EndsWith/IEndsWith/IExact use `Lookup[str]`, all others use `Lookup[Any]`
- Double negation (`~~pred`) produces `Not(Not(pred))` -- no simplification (per plan: "compiler handles semantics")
