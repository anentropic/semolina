  <behavior>
    SnowflakeEngine.introspect(view_name: str) -> IntrospectedView:
    1. Execute: SHOW COLUMNS IN SEMANTIC VIEW {view_name}
    2. Extract column names from cur.description (lowercase each for safe access)
    3. For each row, build dict from zip(columns, row, strict=True)
    4. field_type = d["kind"].lower()  # "metric" | "dimension" | "fact"
    5. type_json = json.loads(d["data_type"])
    6. py_type = snowflake_json_type_to_python(type_json)  # str or None
    7. data_type = f"TODO: {d['data_type']}" if py_type is None else py_type
    8. description = d.get("comment", "") or ""
    9. Return IntrospectedView(view_name, class_name=_to_pascal_case(view_name), fields=[...])
    On ProgrammingError/DatabaseError: raise RuntimeError(f"Snowflake introspection failed: {e}")

    DatabricksEngine.introspect(view_name: str) -> IntrospectedView:
    1. Execute: DESCRIBE TABLE EXTENDED {view_name} AS JSON
    2. row = cur.fetchone()
    3. schema = json.loads(row[0])
    4. For each col in schema["columns"]:
       - is_measure = col.get("is_measure", False)
       - field_type = "metric" if is_measure else "dimension"
       - type_obj = col.get("type", {})
       - py_type = databricks_type_to_python(type_obj if isinstance(type_obj, dict) else {"name": str(type_obj)})
       - data_type = f"TODO: {type_obj}" if py_type is None else py_type
       - description = col.get("comment", "") or ""
    5. Return IntrospectedView(view_name, class_name=_to_pascal_case(view_name), fields=[...])
    On DatabaseError/OperationalError/Error: raise RuntimeError(f"Databricks introspection failed: {e}")

    _to_pascal_case(view_name: str) -> str:
    - Extract last segment after final "." (handles schema.view and catalog.schema.view)
    - Split by "_", capitalize each word, join: "sales_revenue_view" -> "SalesRevenueView"
    - This helper is module-level private in each engine file (or a shared utility)

    Test strategy: Use unittest.mock.patch to mock connector.connect().__enter__().cursor().__enter__() chain.
    SnowflakeEngine tests: patch "snowflake.connector.connect" with a MagicMock that returns mock cursor.
    DatabricksEngine tests: patch "databricks.sql.connect" with a MagicMock.
    Tests verify: correct SQL sent to cursor.execute(), correct IntrospectedView returned, error handling.
