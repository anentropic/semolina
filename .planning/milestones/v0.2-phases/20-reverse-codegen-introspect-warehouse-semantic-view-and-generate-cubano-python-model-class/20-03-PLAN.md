  <behavior>
    render_views(views: list[IntrospectedView]) -> str:
    - Renders all views into a single Python source string
    - Single shared imports section at top: "from cubano import SemanticView, Metric, Dimension, Fact"
    - If ANY field across ALL views has data_type in ("datetime.date", "datetime.datetime", "datetime.time"):
      emit "import datetime" after the cubano import
    - For each view, emit:
        class {class_name}(SemanticView, view="{view_name}"):
            (class body - fields in order)
    - No class docstring (warehouse views don't have view-level descriptions — field descriptions only)
    - For each field in view.fields:
        - If field.data_type starts with "TODO:": emit "    # {field.data_type}" before field line
        - Emit "    {field.name} = {field_class}()" where field_class is:
            - "Metric" if field.field_type == "metric"
            - "Fact" if field.field_type == "fact"
            - "Dimension" otherwise
        - If field.description (non-empty): emit '    """{field.description}"""' on next line
    - Blank line between class definitions (standard Python formatting)
    - Returns raw rendered string (before ruff formatting)

    format_with_ruff(source: str) -> str:
    - Runs: subprocess.run(["uv", "run", "ruff", "format", "--stdin-filename", "models.py", "-"], input=source, capture_output=True, text=True, check=False)
    - If returncode == 0: return result.stdout
    - If FileNotFoundError (uv not found): return source unchanged
    - If returncode != 0: return source unchanged (ruff format failure is non-fatal)

    render_and_format(views: list[IntrospectedView]) -> str:
    - Convenience wrapper: render_views(views) then format_with_ruff(result)
    - This is what the CLI calls

    Template structure (python_model.py.jinja2):
    ```
    from cubano import SemanticView, Metric, Dimension, Fact
    {% if needs_datetime %}
    import datetime
    {% endif %}
    {% for model in models %}


    class {{ model.class_name }}(SemanticView, view="{{ model.view_name }}"):
    {% for field in model.fields %}
    {% if field.todo_comment %}
        # {{ field.todo_comment }}
    {% endif %}
        {{ field.name }} = {{ field.field_class }}()
    {% if field.docstring %}
        """{{ field.docstring }}"""
    {% endif %}
    {% endfor %}
    {% endfor %}
    ```

    Test cases:
    - Single view with one metric → "revenue = Metric()"
    - Single view with dimension → "country = Dimension()"
    - Single view with fact → "unit_price = Fact()"
    - Field with description → docstring emitted on next line
    - Field without description → no docstring
    - Field with data_type starting "TODO:" → # comment above field line
    - Field with datetime.date type → "import datetime" at top
    - Field with datetime.datetime type → "import datetime" at top
    - No datetime fields → no "import datetime"
    - Multiple views → single imports section at top, multiple class definitions
    - render_and_format() returns string (integration: run actual ruff if available, else verify source returned unchanged)
    - format_with_ruff() with invalid source → returns source unchanged (ruff exits non-zero)
  </behavior>
