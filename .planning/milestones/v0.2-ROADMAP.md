# Roadmap: Cubano

## Milestones

- ✅ **v0.1 MVP** — Phases 1-7 (shipped 2026-02-16)
- 🚧 **v0.2 Tooling & Documentation** — Phases 8-12 (in progress)

See `.planning/milestones/v0.1-ROADMAP.md` for v0.1 details.

## Overview

Cubano v0.1 shipped a production-ready ORM for data warehouse semantic views. v0.2 adds three complementary capabilities that make Cubano easier to adopt and more reliable in production: integration testing to validate query behavior against real warehouse data, code generation to turn Cubano models into warehouse-native CREATE SEMANTIC VIEW statements, and comprehensive documentation with auto-generated API reference and interactive guides. These features are additive—the core library is unchanged—and ship in a natural dependency order: integration tests first (validate v0.1 stability), codegen second (independent from tests), and documentation third (references both).
