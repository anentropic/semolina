### Implementation Details

**DatabricksEngine Class (228 lines):**

1. **Lazy Import Pattern** (lines 109-117):
   ```python
   try:
       import databricks.sql as _
   except ImportError as e:
       msg = "databricks-sql-connector is required for DatabricksEngine. Install with: pip install cubano[databricks]"
       raise ImportError(msg) from e
   ```

2. **Connection Storage** (line 119):
   - Stores connection params: `self._connection_params = connection_params`
   - No connection created at init time

3. **SQL Generation** (lines 151-152):
   - Delegates to SQLBuilder: `builder = SQLBuilder(self.dialect)`
   - Returns: `builder.build_select(query)`

4. **Execution with Context Managers** (lines 200-212):
   ```python
   with (
       databricks.sql.connect(**self._connection_params) as conn,
       conn.cursor() as cur,
   ):
       cur.execute(sql)
       description: Any = cur.description
       columns: list[str] = [desc[0] for desc in description]
       rows: Any = cur.fetchall()
       return [dict(zip(columns, row, strict=True)) for row in rows]
   ```

5. **Error Translation** (lines 214-228):
   - Catches OperationalError (connection/permissions)
   - Catches DatabaseError (SQL syntax/objects)
   - Catches Error (generic fallback)
   - All translate to RuntimeError with helpful messages
