# Tutorials

Tutorials walk you through Semolina step by step. Start from the beginning and
follow along -- you'll have a working query by the end.

If you already have semantic views in your warehouse, these tutorials show you
how Semolina mirrors them as typed Python models.

- [Installation](installation.md) -- install Semolina and verify your setup
- [Your first query](first-query.md) -- define a model, build a query, and read results
